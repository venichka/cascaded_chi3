# cascaded_chi3 — TCMT simulations of the cascaded χ²:χ² → χ³ effect

Numerical companion to `Cascaded_chi3_metasurface_note.md/.tex`. Each
script reproduces one phenomenon discussed in the note.

## Quick start

```sh
cd cascaded_chi3
conda run -n ml_base python run_all.py
```

Figures are written to `figures/` as both PNG (150 dpi) and PDF.
Individual figures can be regenerated by running the corresponding
`fig_*.py` directly.

## Files

| file | what it shows | section in note |
| :-- | :-- | :-- |
| `tcmt.py` | core two-mode TCMT integrator (Eqs. 4.1–4.2 of the note) | §4 |
| `analytic.py` | adiabatic-elimination closed forms for `α_eff_casc`, `κ_casc`, etc. | §5 |
| `params.py` | dimensionless defaults + LiNbO₃ reference numbers | §9 |
| `fig_validation.py` | TCMT numerics match `α_eff_casc(Δω)`; simple `−1/Δω` form deviates as `Δω → γ₂/2` | §5(i), §10(1) |
| `fig_sign_flip.py` | sweep `Δω` through 0; cascaded Kerr changes sign | §5 |
| `fig_sum_rule.py` | vary `Q₂`: peak ∝ `Q₂`, FWHM ∝ `1/Q₂`, product constant | §6 |
| `fig_bistability.py` | adiabatic up-/down-ramp of `\|s₊\|²` with blue-detuned pump → hysteresis | §10(2) |
| `fig_saturation.py` | Kerr-only vs full TCMT as `\|βa₁\|/\|Δω\| → 1`; pump depletion | §10(2) |
| `fig_pulse_bandwidth.py` | Gaussian pulse: cascaded phase shrinks for `σ_t γ₂ ≲ 1` | §6 (time-domain face) |

## Conventions

All scripts run in dimensionless units with `β = 1` and `γ₂ = 1`, so
detunings, decay rates and pump amplitudes are all measured in units of
`γ₂`. The mapping back to physical units (LiNbO₃ telecom-band reference)
is exposed in `params.py`.
